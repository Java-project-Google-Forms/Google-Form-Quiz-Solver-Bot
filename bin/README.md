# Google-Form-Quiz-Solver-Bot
Telegram-бот для решения задач в Google Forms и сохранения результатов. Решение основано на LLM.

## Описание
Документы с описанием проекта находятся в каталоге docs/.

## Технологичексий стек
- Язык: Java 25.
- Cборка и управление зависимостями: Gradle.
- База данных:  MongoDB.
- Фреймворк: Spring 7 (WebFlux, Data MongoDB Reactive, Modulith).
- Брокер сообщений: Kafka.
- Развертывание и контейнеризация: 
  - Docker.
  - Docker Compose.
- Тестирование: Junit.
